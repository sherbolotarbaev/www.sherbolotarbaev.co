### ENDPOINTS:

#### Start Lab / Create AWS Account (Platform)

**Method**: `POST` <br />
**Endpoint**: `/aws/:exerciseId` <br />
**Response:**

```jsx
credentials: {
  username: string;
  password: string;
  consoleUrl: string;
  accountId: number;
}
```

#### Check Exercise / Check Test Case URLs (compare actual & expected outputs)

**Method**: `POST` <br />
**Endpoint**: `/aws/:exerciseId/check` <br />
**Response:**

```jsx
{
	submissionStatus: 'TODO' | 'SOLVED' | 'ATTEMPTED',
	testCases: [
    {
      result: 'passed' | 'failed',
      input": string, // Stringfied JSON
      output": string, // Stringfied JSON
      testCaseName: string,
      errorMessage: string,
      actualOutput: string, // Stringfied JSON
      stdoutDiff: [
        {
          added: boolean,
          removed: boolean,
          edited: boolean,
          value: {
            from: string, // Stringfied JSON
            to: string, // Stringfied JSON
          } | string,
          count: number
        }
      ]
    }
	  ...
	]
}
```

### Get AWS Account credentials (Platform)

**Method**: `GET` <br />
**Endpoint**: `/aws/:exerciseId` <br />
**Response:**

```jsx
{
	credentials: {
			username: string
			password: string
			consoleUrl: string
			accountId: number
	} | null
}
```

  src="https://www.sherbolotarbaev.co/images/aws-exercise-support3.png"

### Clean-up AWS Member Account (Platform)

**Method**: `DELETE` <br />
**Endpoint**: `/aws/:exerciseId/clean-up` <br />
**Response:**

```jsx
{
  success: boolean;
}
```
